Mínimos y máximos

Un ejemplo

Sea (P) un problema de optimización de la forma

$$\text{(P) min } f(x) \\ \qquad \text{s.a. } g_{i}(x) \leq b_{i} \qquad \forall{i = 1, ..., m} \\ x \in C \subseteq \mathbb{R}^{n}$$

Sea \(D := \{x \in \mathbb{R}^{n}: g_{i}(x) \leq b_{i} \qquad \forall{i = 1, ..., m, x \in C}\}\) el dominio de (P).

Un punto \(x \in D\) se dice mínimo global de \(f\) en \(D\) si \(f(x) \leq f(y) \ \ \forall{y \in D}\).

Si \(f(x) < f(y) \ \ \forall{y \in D \ \backslash \ \{x\}}, \ x\) es mínimo estricto.
